# 🎮 언리얼 엔진 블루프린트(Blueprint) 핵심 요약

## 📌 1. 블루프린트 개요
* **정의:** 블루프린트는 C++로 미리 빌드된 함수 및 기능을 언리얼 엔진 내에서 시각적인 **노드(Node) 형태**로 제공하는 비주얼 스크립팅 시스템입니다.
* **노드 종류:** 블루프린트 그래프 내에는 변수 노드, 제어 노드, FK/IK 솔버, 컨트롤 리그(Control Rig) 노드 및 기타 다양한 연산 함수문이 노드 형태로 존재합니다.
* **통신 체계:** 노드와 노드 사이를 와이어(Wire)로 연결함으로써 실행 흐름을 제어하고 객체의 상태값을 정의합니다.

## 📌 3. 변수 접근 제한자 및 변수 속성

블루프린트 디테일 패널에서 제어할 수 있는 변수의 특수 노출 설정입니다.

### 🔹 퍼블릭(Public) & 프라이빗(Private)
* **Public:** 외부의 다른 블루프린트나 에디터의 레벨 디테일 창에서 변수 값을 직접 수정할 수 있도록 개방합니다.
* **Private:** 오직 해당 블루프린트 내부의 로직을 통해서만 접근 및 수정이 가능하도록 제한하여 데이터를 보호합니다.

### 🔹 스폰 시 노출 (Expose on Spawn)
* 특정 상황에 객체가 생성되는 시점(Spawn)에 상태값을 동적으로 직접 주입하여 변수를 초기화할 수 있도록 노드 핀을 열어주는 기능입니다.

### 🔹 시네마틱에 노출 (Expose to Cinematics)
* 말 그대로 컷신(Cutscene)이나 타임라인에 따른 상태의 변화를 만들 때 사용하는 애니메이션 기능인 **시퀀서(Sequencer)** 툴 내에서 해당 변수가 제어될 수 있도록 트랙 노출을 허용합니다.
